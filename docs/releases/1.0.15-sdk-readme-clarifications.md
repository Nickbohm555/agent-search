# agent-search-core 1.0.15

`agent-search-core` `1.0.15` is a documentation-only patch release for the PyPI package.

## README updates

- Removes the section that described reading `decomposition_sub_questions` from an async status payload in the core SDK README.
- Expands the `rerank_enabled` and `query_expansion_enabled` documentation with concrete examples.
- Adds short guidance on what each toggle does and when disabling it is useful.

## Runtime behavior

- No runtime code changes relative to `1.0.14`.
- This release exists so the PyPI project page reflects the intended sync-SDK usage more clearly.
