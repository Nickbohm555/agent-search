# agent-search-core 1.0.16

`agent-search-core` `1.0.16` is a documentation-only patch release for citation metadata setup.

## README updates

- Adds a dedicated citation metadata section to the PyPI README.
- Documents the exact metadata keys the SDK expects for citation construction:
  - `citation_title`
  - `citation_source`
  - `document_id`
- Clarifies that citation content comes from `Document.page_content`.
- Adds a concrete example showing how to store citation metadata on pgvector-backed chunks.

## Runtime behavior

- No runtime code changes relative to `1.0.15`.
- This release exists so PyPI documents the citation contract precisely.
